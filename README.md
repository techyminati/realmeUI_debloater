# realmeUI Debloater

Simple Shell Script that patches realmeUI 2.0 firmware & nukes oplus bloats from it !

## Requirements
* Linux
* Internet Connection
* Python3 

## Usage
``` ./rui_lite.sh <Direct Link to realmeUi 2.0 full OTA> <Device Name> ```
